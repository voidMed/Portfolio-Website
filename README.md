# Portfolio Website

[![CI](https://github.com/voidMed/Portfolio-Website/actions/workflows/ci.yml/badge.svg)](https://github.com/voidMed/Portfolio-Website/actions/workflows/ci.yml)

Personal portfolio built with **React** and **Vite**.

## Sections

- Hero
- About
- Skills
- Projects
- Experience
- Education
- Contact

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
